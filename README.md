# Paint Pad

A small browser paint app inspired by classic Windows Paint. It supports drawing, airbrush/watercolor/oil brushes, a top-left Markers button with calligraphy pen/crayon/pencil plus size and opacity controls, painting the canvas, erasing, magnifying, undo, clearing, color selection, editing a custom color, setting the background color, a 2G Songs shortcut to Google Gemini, and brush size. It intentionally has no text tool and no text boxes.

## Run

Open `index.html` in any modern browser.

## Test

1. Draw on the canvas with the brush selected.
2. Use the top-left `Markers` button to choose Calligraphy pen, Crayon, or Pencil, then adjust Size and Opacity.
3. Use the bottom-right toolbar to switch colors, open Color Options, open 2G Songs, magnify, fill the canvas, erase, undo, and clear.
4. Use the bottom-left `Background` button to set the canvas background to the active color.

The page also exposes `window.paintPadTestApi` for browser-console checks.
